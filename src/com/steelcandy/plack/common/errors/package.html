This package contains classes that are used in error handling in most or all language implementations. Most of the classes are either
  • subclasses of PlackError and represent an error that was discovered while processing a language's source code, or
  • subclasses of ErrorHandler and represent classes that handle the errors that occur while processing a language's source code.
There are a few notable exceptions:
  • ErrorSeverityLevels is an interface that defines constants that represent the various severity levels that an error can have
  • FatalErrorException is the class of (runtime) exception used to stop processing immediately when a fatal error is discovered while processing a language's source code
