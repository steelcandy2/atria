This package contains classes that represent objects having to do with pieces of source code. Most of the classes fall into one of the following groups:
  • SourceCode and its subclasses, which represent pieces of source code: SourceCodeFile represents a piece of source code in a file, and SourceCodeString represents a piece of source code in a Java String
  • SourcePosition and SourceLocation, which represent the position of a single character and the location of a range of characters in a piece of source code, respectively; and SourceLocationFactory and its subclasses, which create various types of SourceLocations
  • CharacterIterator and its subclasses, which represent iterators over characters
In addition, the HasSourceLocation interface is implemented by classes that represent objects that have a location in a piece of source code, such as tokens and constructs for example.
