This package contains classes that represent objects having to do with breaking source code up into tokens. Most of the classes fall into one of the following groups:
  • Token and its subclasses, which represent individual tokens
  • Tokenizer and its subclasses, which represent convert their input into Tokens:
    • SourceCodeTokenizer and its subclasses convert a piece of source code into Tokens
    • FilterTokenizer and its subclasses convert the output of another Tokenizer into (usually different) Tokens
  • TokenCreator and its subclasses, which create zero or more tokens from source code
  • TokenCreatorResult and its subclasses, which represent the results of TokenCreators creating tokens
In addition, TokenFactory can create (vie reflection) various types of Tokenizers, and TokenSinks will extract all of the tokens from a Tokenizer (which is mainly useful in testing Tokenizers).

Various token-related collection, iterator and predicate classes are also defined in this package.
